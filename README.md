# 📝 To-Do App

A simple To-Do application built with **HTML, CSS, and JavaScript**.  
It allows users to add, delete, and manage daily tasks in a clean interface.

---

## 🚀 Features
- Add new tasks  
- Mark tasks as complete  
- Delete tasks  
- User-friendly interface
